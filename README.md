# mojn-ios 

*Meaning: Mojn [ˈmʌjn] means both hello and goodbye. It can be used at all times of a day.*



## Preliminary
Project is using [Carthage](https://github.com/Carthage/Carthage).
To fetch dependencies:

```sh

carthage update --platform ios

```
